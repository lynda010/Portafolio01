Proyecto actual (2025)

Actualmente participo en el desarrollo de un software a medida para una academia de capacitación en seguridad privada. El objetivo es optimizar la administración de cursos, instructores, estudiantes y procesos internos.

En este proyecto utilizo:

  • HTML5 y CSS3 para la estructura y el diseño visual.
  • JavaScript para dinamismo y validación de datos.
  • PHP como lenguaje de servidor para la lógica y conexión con la base de datos.
  • MySQL para la gestión de información académica y administrativa.
  • Bootstrap para un diseño atractivo y adaptable.
  • Visual Studio Code como editor principal de desarrollo.
  • GitHub para versionar y dar seguimiento al progreso del sistema.

Principales funcionalidades desarrolladas:

  • Registro y gestión de estudiantes.
  • Control de asistencia.
  • Administración de cursos y programación de horarios.
  • Asignación de instructores.
  • Generación de certificados y reportes automatizados.

Este proyecto me ha brindado la oportunidad de poner en práctica conocimientos técnicos en un contexto real, así como mejorar mi capacidad de análisis, organización y solución de problemas.
